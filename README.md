## APIs

### Charter Communications Spectrum Enterprise API

The Spectrum Enterprise Open API exposes REST APIs providing enterprise clients the ability to integrate their systems directly with Spectrum Enterprise portal features, including service ticketing and carrier serviceability. The API supports OAuth 2.0 authentication.

- **Human URL:** https://enterprise.spectrum.com/
- **Tags:** Telecommunications, Enterprise, Networking, Ticketing
- **Properties:**

### Charter Communications Bryte IQ API

Bryte IQ is a Network-as-a-Service (NaaS) API platform launched by Charter Communications in collaboration with CableLabs. It provides secure, privacy-friendly access for developers based on the Linux Foundation CAMARA project, enabling visibility into connected devices and network capabilities.

- **Human URL:** https://corporate.charter.com/newsroom/charter-and-cablelabs-launch-bryte-iq-network-as-a-service-platform
- **Tags:** Network as a Service, NaaS, Telecommunications, CAMARA
